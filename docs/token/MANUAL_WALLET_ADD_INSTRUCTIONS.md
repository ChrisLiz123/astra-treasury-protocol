# AstraToken Manual Wallet Add Instructions

## Base Mainnet

Network: Base Mainnet

Token name: AstraTreasury Token

Contract address: see https://astratreasury.ai/api/public/token

## Rule

Adding ASTP to a wallet only lets the wallet display the token. It does not approve buying, selling, staking, rewards, or treasury funding.
