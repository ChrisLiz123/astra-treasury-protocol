# DEX Parameter Finalization Boundaries

## Review can do

- Determine whether a selected parameter set is ready for later approval.
- Identify missing parameter fields.
- Preserve restricted-mode controls.
- Prepare for an action-specific parameter approval package.

## Review cannot do

- Approve parameter finalization.
- Create a liquidity pool.
- Add liquidity.
- Move funds.
- Generate a Safe payload.
- Execute a Safe transaction.
- Publish a public trading link.
- Activate a buy page.
- Approve full launch.

## Rule

Review readiness is not public trading approval.
