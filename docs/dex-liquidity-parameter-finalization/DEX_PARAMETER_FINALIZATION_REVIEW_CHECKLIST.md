# DEX Parameter Finalization Review Checklist

## Required before later parameter approval

- [ ] DEX/liquidity path selected.
- [ ] Parameter review gate ready.
- [ ] Parameter selection imported, if applicable.
- [ ] Pair selected.
- [ ] Pool version selected.
- [ ] Fee tier or pool type selected.
- [ ] Initial price approach selected.
- [ ] Liquidity amounts selected.
- [ ] Liquidity source selected.
- [ ] Risk disclosures drafted.
- [ ] Safe/treasury impact reviewed.
- [ ] Public trading link plan drafted.
- [ ] Buy page language drafted.

## Still not allowed

- Pool creation.
- Liquidity provisioning.
- Safe payload generation.
- Safe transaction execution.
- Public trading activation.
- Buy page activation.
- Treasury funds movement.
- Full launch approval.

## Rule

Passing this review only prepares the project for a later action-specific parameter approval step.
