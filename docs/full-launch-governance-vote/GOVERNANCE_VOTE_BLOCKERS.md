# Governance Vote Blockers

The following block governance vote opening:

- Final governance approver not assigned.
- Vote scope not approved.
- Resolution text not approved.
- Capability matrix not reviewed.
- Public status update not drafted.
- Voter or signer list not confirmed.
- Monitoring status not reviewed.
- Incident status not reviewed.

The following block governance approval recording:

- Vote not opened.
- Vote not completed.
- Resolution not signed.
- Result not recorded.
- Evidence snapshot not archived.
- Public status not updated.

## Rule

Governance vote preparation is not governance approval.
