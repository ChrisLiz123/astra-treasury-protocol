# Governance Vote Checklist

## Before opening a vote

- [ ] Vote scope reviewed.
- [ ] Resolution text reviewed.
- [ ] Capability matrix reviewed.
- [ ] Treasury funding implications reviewed.
- [ ] Execution queue implications reviewed.
- [ ] Public disclosure implications reviewed.
- [ ] Operator runbook reviewed.
- [ ] Monitoring status reviewed.
- [ ] Incident status reviewed.
- [ ] Public status update reviewed.
- [ ] Voter or signer list confirmed.
- [ ] Final governance approver assigned.

## During vote or resolution

- [ ] Vote opened or resolution circulated.
- [ ] Vote/result threshold confirmed.
- [ ] Voter/signature evidence captured.
- [ ] Final decision recorded.
- [ ] Public status updated.
- [ ] Evidence snapshot archived.

## Rule

Do not mark governance approval recorded until the vote/result evidence exists.
