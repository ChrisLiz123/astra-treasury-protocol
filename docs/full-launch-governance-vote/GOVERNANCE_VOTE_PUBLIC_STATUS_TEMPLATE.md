# Governance Vote Public Status Template

## If vote is not opened

Full-launch governance vote package is prepared for review. No vote is open. Full launch and restricted capabilities are not approved.

## If vote is opened later

A public status update must show:

- vote scope
- opened date
- capabilities under consideration
- disabled capabilities
- expected next step

## If vote completes later

A public status update must show:

- vote result
- exact approved capabilities
- exact disabled capabilities
- evidence reference
- next activation milestone

## Rule

Public language must not imply approval before the governance result is recorded.
