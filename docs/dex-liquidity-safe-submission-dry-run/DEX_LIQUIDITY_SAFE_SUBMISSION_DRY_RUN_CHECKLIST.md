# DEX Liquidity Safe Submission Dry Run Checklist

## Required checks

- [x] Liquidity Safe submission preparation complete.
- [x] Liquidity Safe submission approval recorded.
- [x] Liquidity Safe payload generated.
- [x] Liquidity Safe payload verified.
- [x] Safe Transaction Service reachable.
- [x] Liquidity Safe exists.
- [x] No duplicate matching pending transaction.
- [x] Live balances and allowances verified.
- [x] Pool liquidity remains zero.
- [x] Liquidity Safe transaction not submitted.
- [x] Liquidity Safe transaction not executed.
- [x] Liquidity not added.
- [x] Position not minted.
- [x] Public trading not approved.
- [x] Full launch not approved.

## Required after dry run

- [ ] Liquidity Safe submission live.
- [ ] Pending signatures monitoring.
- [ ] Liquidity Safe execution approval.

## Rule

Do not submit or execute the Safe payload during this dry-run milestone.
