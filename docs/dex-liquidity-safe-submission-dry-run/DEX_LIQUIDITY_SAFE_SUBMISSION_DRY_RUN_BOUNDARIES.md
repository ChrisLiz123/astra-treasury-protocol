# DEX Liquidity Safe Submission Dry Run Boundaries

## This package can do

- Validate readiness for Safe submission.
- Check Safe Transaction Service.
- Confirm the liquidity Safe exists.
- Confirm no matching pending transaction already exists.
- Publish dry-run evidence.

## This package cannot do

- Submit a liquidity Safe transaction.
- Execute a liquidity Safe transaction.
- Add liquidity.
- Mint a position on-chain.
- Enable public trading.
- Activate the buy page.

## Rule

Dry-run only.
