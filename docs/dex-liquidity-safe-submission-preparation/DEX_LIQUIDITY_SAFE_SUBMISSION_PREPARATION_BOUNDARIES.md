# DEX Liquidity Safe Submission Preparation Boundaries

## This package can do

- Prepare Safe UI submission instructions.
- Prepare Transaction Builder import instructions.
- Record operator warnings.
- Publish preparation status.

## This package cannot do

- Submit a liquidity Safe transaction.
- Execute a liquidity Safe transaction.
- Add liquidity.
- Mint a position on-chain.
- Enable public trading.
- Activate the buy page.

## Rule

Preparation only.
