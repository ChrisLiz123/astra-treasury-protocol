# DEX Liquidity Funding Transfer Safe Execution Approval Boundaries

## This package can do

- Record approval for later execution.
- Confirm Safe threshold is reached.
- Confirm the Safe transaction is still not executed.
- Confirm no funds have moved.
- Publish approval evidence.

## This package cannot do

- Execute the Safe transaction.
- Move treasury funds.
- Execute token approvals.
- Add liquidity.
- Mint a position.
- Enable public trading.
- Activate the buy page.

## Rule

Approval-only.
