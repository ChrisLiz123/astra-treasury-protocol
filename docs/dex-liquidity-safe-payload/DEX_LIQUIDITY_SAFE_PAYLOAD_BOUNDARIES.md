# DEX Liquidity Safe Payload Boundaries

## This package can do

- Generate a local Safe payload.
- Generate a Safe Transaction Builder JSON file.
- Record payload hash and Transaction Builder hash.
- Publish payload generation status.

## This package cannot do

- Submit a liquidity Safe transaction.
- Execute a liquidity Safe transaction.
- Add liquidity.
- Mint a position on-chain.
- Enable public trading.
- Activate the buy page.

## Rule

Payload generation only.
