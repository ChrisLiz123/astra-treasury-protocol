# Governance Vote Evidence Plan

## Purpose

Define what evidence must be preserved if a governance vote is opened.

## Required evidence

- Pre-vote public status snapshot.
- Vote scope document.
- Resolution text.
- Capability activation matrix.
- Public notice.
- Vote opening timestamp.
- Vote closing timestamp.
- Vote result, if completed.
- Signer/voter evidence, where appropriate.
- Final decision record.
- Post-vote public status snapshot.

## Evidence archive

Evidence snapshots should be captured before opening, after opening, after closing, and after public status updates.

## Rule

No governance result should be treated as final without evidence.
