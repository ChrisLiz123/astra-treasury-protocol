# Governance Vote Notice Template

## Draft notice

AstraTreasury has prepared a full-launch governance vote package for review.

No governance vote is currently open. Full launch is not approved. Public token sale, real treasury funding, staking/rewards, buybacks, autonomous execution, paper-to-on-chain automation, and mainnet execution queue activation remain disabled.

## If a vote is opened later

The notice must include:

- Vote title.
- Vote scope.
- Open time.
- Close time.
- Voting mechanism.
- Capabilities under consideration.
- Capabilities that remain disabled.
- Link to public status board.
- Link to Trust Center.
- Link to evidence archive.

## Rule

Public language must not imply approval before a governance result is recorded.
