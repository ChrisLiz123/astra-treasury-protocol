# DEX Liquidity Token Approval Safe Submission Approval Boundaries

## This package can do

- Record approval for later Safe submission/proposal.
- Confirm token approval Safe payload verification is complete.
- Confirm no approvals have executed.
- Publish submission approval status.

## This package cannot do

- Submit the Safe transaction.
- Execute token approvals.
- Generate liquidity calldata.
- Add liquidity.
- Mint a position.
- Enable public trading.
- Activate the buy page.

## Rule

Approval-only.
