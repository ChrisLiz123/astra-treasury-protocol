# DEX Liquidity Token Approval Safe Submission Approval

## Purpose

Record approval to submit/propose the verified DEX liquidity token-approval Safe payload in a later step.

## Current status

Token approval Safe payload generated: yes.

Token approval Safe payload verified: yes.

Token approval Safe submission approval: pending or recorded.

Token approval Safe transaction submitted: no.

Token approval executed: no.

Liquidity calldata generated: no.

Liquidity added: no.

Position minted: no.

Public trading approved: no.

Full launch approved: no.

## What this approval can approve

- Proceeding to a later token approval Safe submission preparation step.
- Submitting/proposing only the verified token-approval Safe payload.
- Using the verified payload hash, Transaction Builder hash, approval spender, token targets, and approval amounts.

## What this approval does not do

- Submit the Safe transaction now.
- Execute token approvals.
- Generate liquidity calldata.
- Add liquidity.
- Mint a position.
- Enable public trading.
- Activate the buy page.
- Approve full launch.

## Rule

Submission approval is not Safe submission and is not approval execution.
