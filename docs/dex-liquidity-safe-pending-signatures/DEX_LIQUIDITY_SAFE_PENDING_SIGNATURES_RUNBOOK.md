# DEX Liquidity Safe Pending Signatures Runbook

## Step 1

Run the pending signatures gate.

## Step 2

Review confirmation count, threshold, and missing confirmation count.

## Step 3

If threshold is not reached, collect additional Safe owner signatures and rerun.

## Step 4

If threshold is reached, proceed to liquidity Safe execution approval.

## Rule

Do not execute until a separate execution approval milestone is complete.
