# DEX Liquidity Safe Pending Signatures Boundaries

## This package can do

- Monitor Safe confirmations.
- Confirm the pending Safe transaction matches the verified payload.
- Publish pending-signature status.

## This package cannot do

- Execute a liquidity Safe transaction.
- Add liquidity.
- Mint a position on-chain.
- Enable public trading.
- Activate the buy page.

## Rule

Monitoring only.
