# Global Treasury Funding Approval Review Checklist

## Required checks

- [x] Full launch live recorded.
- [x] Full launch approved.
- [x] Buy page activated.
- [x] Public trading live.
- [x] Liquidity verified.
- [x] Position minted.
- [x] Pool liquidity greater than zero.
- [x] Position owner is liquidity Safe.
- [x] Treasury funding not approved.
- [x] Treasury funding not executed.
- [x] Funds not moved.

## Required after review

- [ ] Global treasury funding approval.
- [ ] Funding payload generation only after separate approval.
- [ ] Funding transfer execution only after separate approval and Safe execution path.

## Rule

Do not generate funding payloads or move funds during this review milestone.
