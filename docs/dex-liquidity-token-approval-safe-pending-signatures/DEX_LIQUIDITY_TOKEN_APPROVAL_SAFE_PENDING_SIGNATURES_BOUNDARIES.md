# DEX Liquidity Token Approval Safe Pending Signatures Boundaries

## This package can do

- Monitor Safe confirmations.
- Record pending-signature state.
- Confirm the Safe transaction is not executed.
- Confirm token approvals have not executed.
- Publish monitoring evidence.

## This package cannot do

- Execute the Safe transaction.
- Execute token approvals.
- Generate liquidity calldata.
- Add liquidity.
- Mint a position.
- Enable public trading.
- Activate the buy page.

## Rule

Monitoring only.
