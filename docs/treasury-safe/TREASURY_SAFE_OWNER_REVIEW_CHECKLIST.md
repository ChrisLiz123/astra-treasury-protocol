# Treasury Safe Owner Review Checklist

## Identity checks

- [ ] Treasury Safe address verified.
- [ ] Governance Safe address verified.
- [ ] Destination address verified.
- [ ] Sender wallet or Safe verified.
- [ ] Asset address verified, if tokenized.

## Funding checks

- [ ] Funding source approved.
- [ ] Source of funds documented.
- [ ] Funding asset approved.
- [ ] Funding amount within approved limits.
- [ ] Treasury risk limits approved.
- [ ] Accounting review complete.
- [ ] Compliance review complete.

## Transaction checks

- [ ] Transaction payload reviewed.
- [ ] Destination verified against approved destination.
- [ ] Asset and amount match approved decision record.
- [ ] No extra transactions included.
- [ ] Safe nonce reviewed.
- [ ] Signer threshold confirmed.
- [ ] Execution timing reviewed.
- [ ] Rollback or incident process reviewed.

## Rule

Do not sign a Treasury Safe transaction unless every item is complete.
