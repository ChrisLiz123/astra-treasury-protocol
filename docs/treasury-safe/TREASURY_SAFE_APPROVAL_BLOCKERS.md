# Treasury Safe Approval Blockers

The following block Treasury Safe approval:

- Treasury risk limits not approved.
- Funding source not approved.
- Source of funds not documented.
- Public disclosures not approved.
- Funding transaction dry run incomplete.
- Governance approval missing.
- Incident response process not current.
- Mainnet monitor not passing.
- Active incidents present.
- Transaction payload not reviewed.
- Destination, asset, or amount mismatch.
- Safe owners unavailable or uncertain.

## Rule

Treasury Safe approval must remain blocked until all prerequisites are complete.
