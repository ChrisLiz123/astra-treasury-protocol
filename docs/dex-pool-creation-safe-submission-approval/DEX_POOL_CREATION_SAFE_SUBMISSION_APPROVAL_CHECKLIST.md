# DEX Pool Creation Safe Submission Approval Checklist

## Required before recording approval

- [x] Payload verification review completed.
- [x] Local Safe payload generated.
- [x] Safe payload generation approval recorded.
- [x] Fresh no-pool recheck still shows no selected pool.
- [x] Safe owners and threshold review completed.
- [x] Factory/router execution path review completed.
- [x] Token ordering and sqrtPriceX96 review completed.
- [x] Mainnet monitor passing.
- [x] Active incidents zero.
- [x] Execution queue disabled.
- [x] Treasury funding not approved.
- [x] Safe transaction not submitted.
- [x] Safe transaction not executed.
- [x] Pool not created.
- [x] Liquidity not added.
- [x] Funds not moved.
- [x] Full launch not approved.

## Required before actual Safe submission

- [ ] Operator Safe submission command reviewed.
- [ ] Public status update prepared.
- [ ] Safe execution approval recorded separately.

## Rule

Recording approval does not submit the Safe transaction.
