# DEX Liquidity Token Approval Post-Execution Allowance Verification Boundaries

## This package can do

- Verify current token allowances.
- Verify current token balances.
- Verify token approval execution evidence.
- Publish allowance verification status.

## This package cannot do

- Generate liquidity calldata.
- Add liquidity.
- Mint a position.
- Enable public trading.
- Activate the buy page.
- Approve full launch.

## Rule

Verification only.
