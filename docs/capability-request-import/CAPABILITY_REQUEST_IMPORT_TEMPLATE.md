# Capability Request Import Template

## Purpose

Provide a guarded way to import a future capability activation request for review.

## Current approved mode

Restricted Base Mainnet operation.

## Governance state

Governance decision is recorded for restricted-mode / all-disabled posture.

## Approved capabilities

None.

## Disabled / not approved

- Full launch.
- Public token sale.
- Real treasury funding.
- Staking or rewards.
- Buyback program.
- Mainnet execution queue activation.
- Paper-to-on-chain automation.
- Autonomous execution.
- Safe transaction payload generation.
- Safe transaction execution.

## Import rule

Importing a request does not approve the request.

A request import only creates a review item for a future action-specific approval path.
