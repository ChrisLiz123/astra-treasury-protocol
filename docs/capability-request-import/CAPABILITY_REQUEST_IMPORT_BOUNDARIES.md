# Capability Request Import Boundaries

## Import does not authorize

- Full launch.
- Public token sale.
- Treasury funding.
- Staking or rewards.
- Buybacks.
- Execution queue activation.
- Paper-to-on-chain automation.
- Autonomous execution.
- Safe payload generation.
- Safe transaction execution.

## Import requires later review

A valid imported request must still go through a separate action-specific review and approval path.

## Rule

Do not use request import as an approval record.
