# Capability Request Import Runbook

## Step 1

Confirm the requested capability is exactly one allowed capability.

## Step 2

Confirm restricted-mode gates still pass.

## Step 3

Confirm requester, authority, evidence, and public status plan are present.

## Step 4

Import the request using the guarded importer.

## Step 5

Run capability intake and request import gates.

## Step 6

If the request is valid, create a separate action-specific review path.

## Rule

The import step must not approve, activate, fund, execute, or generate any Safe payload.
