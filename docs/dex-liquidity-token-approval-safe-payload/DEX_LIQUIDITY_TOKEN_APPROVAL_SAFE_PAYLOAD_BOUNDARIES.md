# DEX Liquidity Token Approval Safe Payload Boundaries

## This package can do

- Generate ERC-20 approve calldata.
- Generate a local Safe payload candidate.
- Generate Safe Transaction Builder JSON.
- Publish payload status.

## This package cannot do

- Submit the Safe transaction.
- Execute token approvals.
- Generate liquidity mint calldata.
- Add liquidity.
- Mint a position.
- Enable public trading.
- Activate the buy page.
- Approve full launch.

## Rule

Local payload only.
