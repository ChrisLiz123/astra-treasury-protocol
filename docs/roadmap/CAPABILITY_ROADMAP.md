# AstraTreasury Capability Roadmap

## Current mode

AstraTreasury is deployed on Base Mainnet in restricted operational mode.

## Current status

Enabled:

- Public contract visibility.
- Public transparency pages.
- Mainnet monitor.
- Event monitor.
- Alert escalation.
- Incident acknowledgement.
- Restricted launch reporting.

Disabled:

- Public token sale.
- Real treasury funding.
- Staking or rewards.
- Buyback program.
- Mainnet execution queue.
- Paper-to-on-chain automation.
- Autonomous execution.

## Roadmap principle

Every future capability requires a separate approval milestone.

No future capability is enabled by this roadmap.
