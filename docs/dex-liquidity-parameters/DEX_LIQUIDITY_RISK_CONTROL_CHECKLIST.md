# DEX Liquidity Risk Control Checklist

## Required reviews

- [ ] Initial price reviewed.
- [ ] Liquidity depth reviewed.
- [ ] Slippage risk reviewed.
- [ ] Impermanent loss risk reviewed.
- [ ] MEV/front-running risk reviewed.
- [ ] Token impersonation risk reviewed.
- [ ] Incorrect pair risk reviewed.
- [ ] Public communications reviewed.
- [ ] Legal/compliance final check completed.
- [ ] Treasury asset use reviewed.
- [ ] Safe transaction path reviewed, if applicable.

## Rule

Risk review does not approve public trading.
