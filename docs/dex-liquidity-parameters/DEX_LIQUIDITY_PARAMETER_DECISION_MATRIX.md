# DEX Liquidity Parameter Decision Matrix

## Decisions

| Decision | Status | Notes |
|---|---:|---|
| DEX venue | Pending | Candidate: Uniswap on Base |
| Pool version/type | Pending | Review v3/v4 availability and operational fit |
| Token pair | Pending | Candidate: ASTRA/WETH or ASTRA/USDC |
| Fee tier/pool type | Pending | Requires volatility and liquidity review |
| Initial price approach | Pending | Must be approved before pool creation |
| Initial liquidity amount | Pending | Must be approved before funding |
| Liquidity source | Pending | Must identify whether treasury assets are used |
| Price range | Pending | Required for concentrated-liquidity approach |
| Public risk disclosure | Pending | Required before public trading link |
| Safe transaction path | Pending | Required if Safe assets are used |
| Public trading link | Pending | Must not be published before approval |

## Rule

No parameter is final until separately approved.
