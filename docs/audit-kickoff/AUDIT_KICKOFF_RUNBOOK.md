# Audit Kickoff Runbook

## Purpose

Prepare AstraTreasury Protocol v0.1.1 for external audit kickoff.

## Status

Mainnet launched: no
Real treasury funds: no
Public token sale: no
Audit kickoff readiness: yes

## Kickoff prerequisites

- Auditor selected.
- Scope confirmed.
- Statement of work confirmed.
- Audit start tag created.
- Audit branch created.
- Findings tracker ready.
- Remediation workflow ready.
- Retest workflow ready.
- Communication channel ready.

## Audit-start rule

After the audit-start tag is created, contract changes should happen through clearly named remediation branches.

## No-mainnet rule

Audit kickoff does not authorize mainnet deployment.
