# Finding ID Registry

## Format

ASTRA-001, ASTRA-002, ASTRA-003, and so on.

## Registry

| ID | Source | Severity | Title | Status | Notes |
|---|---|---|---|---|---|
| ASTRA-001 | Internal | MEDIUM | Cancelled signals executable in v0.1.0 | FIX_VERIFIED | Fixed in v0.1.1 before external audit |

## Status values

- OPEN
- ACKNOWLEDGED
- FIX_IN_PROGRESS
- FIXED_PENDING_RETEST
- FIX_VERIFIED
- WONT_FIX_ACCEPTED_RISK

## Rule

Every external audit finding must be added to this registry and to the findings tracker.
