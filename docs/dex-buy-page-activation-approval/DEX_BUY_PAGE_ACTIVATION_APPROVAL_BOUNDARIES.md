# DEX Buy Page Activation Approval Boundaries

## This package can do

- Record approval for later buy page activation.
- Confirm public trading and trading link approval already exist.
- Confirm liquidity and position evidence.
- Confirm full launch remains off.
- Publish approval status.

## This package cannot do

- Activate the buy page.
- Approve global treasury funding.
- Approve full launch.

## Rule

Approval only.
