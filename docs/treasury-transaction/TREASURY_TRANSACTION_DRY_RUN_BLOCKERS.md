# Treasury Transaction Dry Run Blockers

The following block transaction preparation:

- Treasury risk limits not approved.
- Funding source not approved.
- Treasury Safe approval not recorded.
- Public disclosures not final-approved.
- Governance approval missing.
- Source of funds not documented.
- Destination not approved.
- Asset not approved.
- Amount not approved.
- Safe transaction payload not reviewed.
- Incident response process not current.

## Rule

No transaction should be prepared while any blocker remains.
