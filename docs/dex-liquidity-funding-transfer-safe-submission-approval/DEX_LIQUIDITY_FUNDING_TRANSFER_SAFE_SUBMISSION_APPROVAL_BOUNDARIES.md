# DEX Liquidity Funding Transfer Safe Submission Approval Boundaries

## This package can do

- Record approval for a later Safe submission/proposal step.
- Confirm payload verification is complete.
- Confirm no funds have moved.
- Preserve public evidence.

## This package cannot do

- Submit the Safe transaction.
- Execute the Safe transaction.
- Move treasury funds.
- Execute token approvals.
- Add liquidity.
- Mint a position.
- Enable public trading.
- Activate the buy page.

## Rule

Approval-only.
