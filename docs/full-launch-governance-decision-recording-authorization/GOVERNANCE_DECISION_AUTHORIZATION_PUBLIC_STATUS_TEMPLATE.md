# Governance Decision Authorization Public Status Template

## If authorization is not recorded

Governance decision recording authorization package is prepared for review. Decision recording is not authorized. No governance decision is recorded. Full launch and restricted capabilities are not approved.

## If authorization is later recorded

A public status update must include:

- authorization date
- authorizer
- decision record reference
- vote result reference
- resolution reference
- exact capabilities under consideration
- exact capabilities still disabled
- evidence reference
- next decision-recording milestone

## Rule

Public language must not imply approval before a governance decision is actually recorded.
