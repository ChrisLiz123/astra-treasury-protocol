# Governance Decision Recording Authorization Checklist

## Required before authorizing decision recording

- [ ] Governance vote authorization recorded.
- [ ] Governance vote opened.
- [ ] Governance vote completed.
- [ ] Vote result recorded.
- [ ] Signed resolution reviewed.
- [ ] Resolution signing authorization reviewed.
- [ ] Capability matrix final-approved.
- [ ] Approved capabilities reviewed.
- [ ] Disabled capabilities reviewed.
- [ ] Public status update reviewed.
- [ ] Evidence reference reviewed.
- [ ] Decision recorder assigned.
- [ ] Monitoring status reviewed.
- [ ] Incident status reviewed.
- [ ] Final decision-recording authorization recorded.

## Rule

Do not authorize decision recording until every checklist item is complete.
