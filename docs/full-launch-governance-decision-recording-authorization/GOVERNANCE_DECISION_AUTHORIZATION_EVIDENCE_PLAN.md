# Governance Decision Authorization Evidence Plan

## Purpose

Define evidence required before governance decision recording can be authorized.

## Required evidence before authorization

- Vote authorization record.
- Vote opening record.
- Vote result.
- Vote result evidence.
- Resolution signing authorization record.
- Signed governance resolution.
- Capability activation matrix.
- Public status update draft.
- Monitoring status.
- Incident status.
- Decision recorder assignment.
- Evidence archive snapshot.

## Required evidence after authorization, if later recorded

- Authorization decision record.
- Authorization timestamp.
- Authorizer evidence.
- Updated public status.
- Evidence archive snapshot.

## Rule

Decision-recording authorization should not be treated as valid without evidence.
