# Governance Decision Recording Authorization Blockers

The following block decision-recording authorization:

- Governance vote authorization not recorded.
- Governance vote not opened.
- Governance vote not completed.
- Vote result not recorded.
- Governance resolution not signed.
- Resolution signing authorization not recorded.
- Capability matrix not final-approved.
- Public status update not final-approved.
- Evidence reference not reviewed.
- Decision recorder not assigned.
- Monitoring status not reviewed.
- Incident status not reviewed.
- Final authorization not recorded.

## Rule

Decision-recording authorization preparation is not authorization.
