# Prohibited Public Claims

The following claims must not be used unless separately reviewed and approved:

- Public sale is live.
- Buy now.
- Investment opportunity.
- Passive income.
- Guaranteed returns.
- Risk-free.
- Treasury is funded.
- Staking is live.
- Rewards are live.
- Buybacks are active.
- Autonomous execution is live.
- Trading bot is live.
- Full launch is complete.
- Yield product.
- Guaranteed buyback.

## Rule

Do not use promotional or investment-product language in public disclosures.
