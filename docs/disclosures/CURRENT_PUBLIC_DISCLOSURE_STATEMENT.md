# Current Public Disclosure Statement

AstraTreasury Protocol is deployed on Base Mainnet in restricted operational mode.

The deployment includes public contract visibility, Safe-based governance controls, read-only monitoring, event monitoring, alert escalation, incident acknowledgement, evidence snapshots, token metadata, and public transparency pages.

AstraTreasury is not offering a public token sale. ASTP is not marketed as an investment product. Real treasury funding, staking/rewards, buybacks, autonomous execution, paper-to-on-chain automation, and mainnet execution queue activation remain disabled.

Market data, if available, is displayed as read-only public information and does not constitute an offer, sale, recommendation, or investment claim.

Wallet visibility features only help compatible wallets display ASTP and do not buy, sell, transfer, stake, or approve the token.
