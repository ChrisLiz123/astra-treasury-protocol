# Token and Market Disclosure

## Token identity

ASTP is the AstraTreasury Token on Base Mainnet.

## Token visibility

AstraTreasury may publish token metadata, logo assets, wallet visibility tools, and token-list metadata.

## Important limitations

Wallet visibility is not a public token sale.

Token metadata is not an investment claim.

Market data, if available, is read-only and does not imply an offer, recommendation, liquidity guarantee, treasury funding, staking/rewards, or buyback program.

## Rule

Token identity and market-data pages must not imply that a public sale or full launch has been approved.
