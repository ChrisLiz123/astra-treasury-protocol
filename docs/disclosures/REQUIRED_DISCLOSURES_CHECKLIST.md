# Required Disclosures Checklist

## Restricted mode

- [ ] State that AstraTreasury is in restricted Base Mainnet operation.
- [ ] State that full launch is not approved.
- [ ] State that the mainnet execution queue remains disabled.

## Token status

- [ ] State that no public token sale is approved.
- [ ] State that ASTP is not marketed as an investment product.
- [ ] State that wallet visibility is not a purchase flow.
- [ ] State that market data is read-only if displayed.

## Treasury status

- [ ] State that real treasury funding is not approved.
- [ ] State that treasury funding requires separate approvals.
- [ ] State that no treasury funding transaction is authorized.

## Execution status

- [ ] State that paper-to-on-chain automation is disabled.
- [ ] State that autonomous execution is disabled.
- [ ] State that execution queue activation requires separate approval.

## Rule

Every disclosure should be reviewed before any future public full-launch statement.
