# DEX Liquidity Mint Calldata Verification Boundaries

## This package can do

- Verify generated calldata.
- Verify decoded mint parameters.
- Verify current balances and allowances.
- Verify no liquidity has been added.
- Publish verification evidence.

## This package cannot do

- Generate a liquidity Safe payload.
- Submit a liquidity Safe transaction.
- Execute a liquidity Safe transaction.
- Add liquidity.
- Mint a position on-chain.
- Enable public trading.
- Activate the buy page.

## Rule

Verification only.
