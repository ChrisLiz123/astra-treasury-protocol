# Next Capability Approvals

Each future capability requires a separate approval milestone.

## Public token sale

Requires legal review update, governance approval, public disclosure update, and explicit launch plan.

## Real treasury funding

Requires Treasury Safe approval, accounting review, risk limits, incident response drill, and public disclosure update.

## Staking or rewards

Requires legal review, tax/accounting review, security review, governance approval, and public disclosure update.

## Buyback program

Requires market-conduct legal review, treasury policy review, governance approval, monitoring plan, and public disclosure update.

## Mainnet execution queue

Requires governance approval, treasury funding approval, execution dry run, monitoring plan, and restricted-operation unlock milestone.

## Autonomous execution

Not approved for v1. Requires separate audit, governance approval, emergency controls, and extensive monitoring.
