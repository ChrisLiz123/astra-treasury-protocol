# Mainnet Contract Verification Checklist

## Purpose

Prepare BaseScan verification before any public mainnet announcement.

## Checklist

- [ ] Compiler version recorded.
- [ ] EVM version recorded.
- [ ] Optimizer settings recorded.
- [ ] Constructor arguments encoded.
- [ ] Standard JSON input generated.
- [ ] TreasuryPolicy verified.
- [ ] TreasuryVault verified.
- [ ] SignalRegistry verified.
- [ ] ExecutionController verified.
- [ ] AstraToken verified.
- [ ] Public deployment manifest updated.
- [ ] Public docs updated.
- [ ] GitHub release updated.

## Rule

Mainnet contracts should be verified before any external announcement.
