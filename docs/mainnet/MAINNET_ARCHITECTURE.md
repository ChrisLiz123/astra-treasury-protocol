## Mainnet principle

AstraTreasury should not launch on mainnet until external audit, legal review, multisig setup, key-management controls, monitoring, and incident response are complete.

## Proposed mainnet workflow

AI paper signal -> human signal approval -> SignalRegistry -> policy-aware execution proposal -> TreasuryPolicy check -> multisig execution approval -> ExecutionController -> TreasuryVault

This document is for design and review. It is not a deployment authorization.
