# Mainnet Go / No-Go Checklist

## Security

- [ ] External audit completed
- [ ] Audit fixes merged
- [ ] Post-remediation review completed
- [ ] CI passing
- [ ] Live testnet invariant checks passing
- [ ] Local stateful checks passing
- [ ] Local randomized fuzz checks passing
- [ ] Emergency pause tested

## Legal

- [ ] Securities-law review completed
- [ ] Commodity and market-conduct review completed
- [ ] Money-transmission review completed
- [ ] Tax/accounting review completed
- [ ] Public disclaimers approved
- [ ] Marketing copy approved

## Operations

- [ ] Governance Safe created
- [ ] Treasury Safe created
- [ ] Executor Safe created
- [ ] Signer roles documented
- [ ] Dedicated Base Mainnet RPC configured
- [ ] Backup RPC configured
- [ ] Monitoring configured
- [ ] Incident response runbook completed

## Launch controls

- [ ] No public sale without legal signoff
- [ ] No guaranteed-return language
- [ ] No hidden treasury wallets
- [ ] No autonomous execution
- [ ] No EOA with direct vault execution authority

## Decision

Mainnet launch is blocked until all items above are complete.
