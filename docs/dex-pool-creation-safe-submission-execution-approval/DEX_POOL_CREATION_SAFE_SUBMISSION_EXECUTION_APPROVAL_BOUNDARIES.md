# DEX Pool Creation Safe Submission Execution Approval Boundaries

## This package can do

- Record approval for the later Safe submission action.
- Carry forward the verified payload hash.
- Carry forward the dry-run data hash.
- Preserve public evidence.

## This package cannot do

- POST to the Safe Transaction Service.
- Open or automate the Safe UI.
- Request signatures.
- Queue a Safe transaction.
- Execute a Safe transaction.
- Create a pool.
- Add liquidity.
- Move funds.
- Enable public trading.
- Activate a buy page.

## Rule

Approval-only.
