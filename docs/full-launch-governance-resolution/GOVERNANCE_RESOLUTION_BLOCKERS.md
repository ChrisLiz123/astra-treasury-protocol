# Governance Resolution Signing Blockers

The following block resolution signing:

- Governance vote authorization not recorded.
- Governance vote not opened.
- Governance vote not completed.
- Vote result not recorded.
- Resolution text not final-approved.
- Capability matrix not final-approved.
- Signer list not confirmed.
- Signing authority not confirmed.
- Evidence reference not reviewed.
- Public status update not reviewed.
- Monitoring status not reviewed.
- Incident status not reviewed.
- Final resolution-signing authorization not recorded.

## Rule

Resolution-signing preparation is not resolution signing.
