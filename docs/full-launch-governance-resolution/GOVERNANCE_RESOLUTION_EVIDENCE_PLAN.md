# Governance Resolution Evidence Plan

## Purpose

Define the evidence required before any governance resolution can be signed.

## Required evidence before signing

- Vote authorization record.
- Vote opening record.
- Vote URL.
- Vote result.
- Voter or signer evidence, where appropriate.
- Final resolution text.
- Capability activation matrix.
- Public status update draft.
- Monitoring status.
- Incident status.
- Evidence archive snapshot.

## Required evidence after signing

- Signed resolution.
- Resolution hash or reference.
- Signer evidence.
- Public status update.
- Evidence archive snapshot.

## Rule

No governance decision should be treated as final without resolution evidence.
