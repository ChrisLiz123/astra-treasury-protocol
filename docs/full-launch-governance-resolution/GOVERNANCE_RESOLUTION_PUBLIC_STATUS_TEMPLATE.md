# Governance Resolution Public Status Template

## If resolution is not signed

Governance resolution signing package is prepared for review. No resolution is signed. No governance decision is recorded. Full launch and restricted capabilities are not approved.

## If resolution is later signed

A public status update must include:

- resolution date
- vote result
- exact approved capabilities
- exact disabled capabilities
- evidence reference
- next activation milestone

## Rule

Public language must not imply approval before a signed governance resolution exists.
