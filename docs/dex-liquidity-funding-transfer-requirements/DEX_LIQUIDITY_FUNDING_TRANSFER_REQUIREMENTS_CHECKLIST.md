# DEX Liquidity Funding Transfer Requirements Checklist

## Required checks

- [x] Scoped DEX liquidity treasury funding approval recorded.
- [x] Mint parameter review completed.
- [x] Token approval requirements review completed.
- [x] Pool post-execution verification completed.
- [x] Pool liquidity remains zero.
- [x] Additional funding required before liquidity.
- [x] Funding shortfalls calculated.
- [x] Destination Safe recorded.
- [x] Funding source reference recorded.
- [x] No funding transfer payload generated.
- [x] No funding transfer executed.
- [x] No token approval executed.
- [x] No liquidity added.
- [x] No position minted.
- [x] No public trading approval.
- [x] No full launch approval.

## Required before funding transfer approval

- [ ] Funding transfer approval recorded.
- [ ] Funding transfer payload generation approval recorded.
- [ ] Operator funding command reviewed.
- [ ] Public status update prepared.

## Rule

Do not move funds during this review.
