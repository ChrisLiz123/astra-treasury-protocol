# DEX Liquidity Funding Transfer Requirements Boundaries

## This package can do

- Calculate token shortfalls.
- Record required funding transfer amounts.
- Record the destination Safe.
- Record a source reference and optional source address.
- Preserve public evidence.

## This package cannot do

- Generate transfer calldata.
- Generate Safe payloads.
- Submit a Safe transaction.
- Execute a transfer.
- Execute token approvals.
- Add liquidity.
- Mint a position.
- Enable public trading.
- Activate the buy page.
- Approve full launch.

## Rule

Read/review only.
