# Full Launch Blockers

The following remain blockers until separately approved:

- Legal full-launch review.
- Governance full-launch approval.
- Treasury funding approval.
- Execution queue dry run.
- Updated incident response drill.
- Updated public disclosure.
- Updated operator runbook.
- Capability-specific approvals.

## Capability blockers

- Public token sale: blocked.
- Real treasury funding: blocked.
- Staking/rewards: blocked.
- Buyback program: blocked.
- Execution queue activation: blocked.
- Autonomous execution: blocked.

## Rule

No capability should be enabled only because restricted launch stabilization completed.
