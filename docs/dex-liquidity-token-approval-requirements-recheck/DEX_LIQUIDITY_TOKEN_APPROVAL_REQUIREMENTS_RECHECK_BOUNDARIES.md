# DEX Liquidity Token Approval Requirements Recheck Boundaries

## This package can do

- Recheck ERC-20 balances.
- Recheck ERC-20 allowances.
- Identify required token approvals.
- Publish recheck status.

## This package cannot do

- Generate token approval Safe payloads.
- Execute token approvals.
- Generate liquidity calldata.
- Add liquidity.
- Mint a position.
- Enable public trading.
- Activate the buy page.
- Approve full launch.

## Rule

Recheck only.
