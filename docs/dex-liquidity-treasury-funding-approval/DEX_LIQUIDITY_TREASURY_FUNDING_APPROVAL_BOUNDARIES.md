# DEX Liquidity Treasury Funding Approval Boundaries

## This package can do

- Record scoped funding approval for liquidity planning.
- Record desired token amounts.
- Record current Safe balance coverage.
- Record whether additional funding is required.

## This package cannot do

- Move funds.
- Generate funding transfer calldata.
- Generate token approval calldata.
- Generate liquidity mint calldata.
- Generate Safe payloads.
- Execute token approvals.
- Add liquidity.
- Mint a position.
- Enable public trading.
- Activate the buy page.

## Rule

Approval-only.
