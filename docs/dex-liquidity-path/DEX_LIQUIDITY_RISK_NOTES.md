# DEX/Liquidity Risk Notes

## Areas to review

- Initial price setting.
- Low-liquidity volatility.
- Slippage.
- Impermanent loss.
- Front-running / MEV.
- Token impersonation.
- Incorrect pair selection.
- Public communications.
- Regulatory/compliance posture.
- Treasury asset custody and Safe approvals.

## Rule

Risk notes are not approval. They define review areas for the later dedicated DEX/liquidity action path.
