# DEX/Liquidity Path Boundaries

## This milestone can do

- Select DEX/liquidity-pool trading as the preferred future public purchase path.
- Identify Uniswap on Base as the candidate venue.
- List pending parameter decisions.
- Require a dedicated future action-specific path.

## This milestone cannot do

- Create a liquidity pool.
- Add liquidity.
- Set an initial trading price.
- Generate a Safe payload.
- Execute a Safe transaction.
- Move funds.
- Activate a public buy page.
- Approve public trading.
- Approve full launch.

## Rule

DEX path selection must remain review-only until a dedicated action-specific approval path is complete.
