# DEX/Liquidity Parameter Checklist

## Required before any pool or liquidity action

- [ ] Final DEX venue selected.
- [ ] Token pair selected.
- [ ] Pool type / fee tier selected.
- [ ] Initial price approved.
- [ ] Liquidity amount approved.
- [ ] Liquidity source approved.
- [ ] Treasury funding impact reviewed.
- [ ] Safe transaction requirements reviewed.
- [ ] Public trading risk disclosure drafted.
- [ ] Legal/compliance final check completed.
- [ ] Dedicated action-specific approval recorded.
- [ ] Public status update prepared.

## Rule

No pool, liquidity, Safe payload, or trading link should be created before all items are complete.
