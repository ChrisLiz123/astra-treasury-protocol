# Launch Control Decision Guide

## Generic preparation status

Generic preparation is complete.

## Required for any real activation

- Explicit governance decision.
- Exact capability named.
- Evidence archived.
- Public status updated.
- Monitoring confirmed.
- Incident status confirmed.
- Safe transaction reviewed, if on-chain action is involved.

## What can stop

Repeated generic prep packages.

## What should happen next

Select the exact capability or action being pursued and run only that activation path.
