# Full Launch Readiness Review Checklist

## Required checks

- [x] DEX buy page activation live recorded.
- [x] Buy page activated.
- [x] Public trading live.
- [x] Public trading approved.
- [x] Public trading link approved.
- [x] DEX liquidity post-execution verification complete.
- [x] Liquidity added.
- [x] Position minted.
- [x] Pool liquidity greater than zero.
- [x] Position owner is liquidity Safe.
- [x] `/buy` page generated.
- [x] Full launch not approved.
- [x] Global treasury funding not approved/executed.

## Required after readiness review

- [ ] Full launch approval.
- [ ] Full launch live/public finalization after separate approval.

## Rule

Do not approve full launch during this readiness milestone.
