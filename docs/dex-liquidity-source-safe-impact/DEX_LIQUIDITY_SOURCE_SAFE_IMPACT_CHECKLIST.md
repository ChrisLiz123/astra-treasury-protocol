# DEX Liquidity Source and Safe Impact Checklist

## Required before recording approval

- [ ] DEX parameter approval is recorded.
- [ ] Valid parameter selection exists.
- [ ] Liquidity source classification is selected.
- [ ] Safe-impact classification is selected.
- [ ] Approver/reference is real and not a placeholder.
- [ ] Approval reference is real and not a placeholder.
- [ ] Mainnet monitor is passing.
- [ ] Active incidents are zero.
- [ ] Execution queue remains disabled.
- [ ] Treasury funding remains not approved.
- [ ] Safe payload remains not generated.
- [ ] Full launch remains not approved.

## Rule

Approval records planning classification only.
