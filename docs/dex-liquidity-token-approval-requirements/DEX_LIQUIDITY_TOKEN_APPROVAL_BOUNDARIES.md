# DEX Liquidity Token Approval Requirements Boundaries

## This package can do

- Read token balances.
- Read token allowances.
- Identify whether token approval payloads will be required later.
- Record the spender and Safe owner of funds.

## This package cannot do

- Generate approval calldata.
- Generate Safe payloads.
- Approve token transfers.
- Add liquidity.
- Mint a position.
- Move treasury funds.
- Enable public trading.
- Activate the buy page.
- Approve full launch.

## Rule

Read-only review.
