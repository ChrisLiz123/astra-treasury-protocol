# Treasury Source Review Blockers

The following block funding source approval:

- Source of funds not documented.
- Legal review incomplete.
- Accounting review incomplete.
- Sanctions/compliance review incomplete.
- Funding asset not approved.
- Sender wallet or Safe not approved.
- Destination wallet or Safe not approved.
- Treasury risk limits not approved.
- Governance approval missing.
- Treasury Safe approval missing.
- Public disclosure update incomplete.
- Funding transaction package missing.

## Rule

A cleared audit/legal posture for the overall project does not by itself approve a specific funding source or funding transaction.
