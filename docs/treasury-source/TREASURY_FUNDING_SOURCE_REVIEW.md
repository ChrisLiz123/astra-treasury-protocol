## Purpose

Define the review process for any future source of real treasury funding.

## Required before approving a source

- Source of funds documented.
- Legal review complete.
- Accounting review complete.
- Sanctions/compliance review complete.
- Funding asset approved.
- Sender wallet or Safe approved.
- Destination wallet or Safe approved.
- Treasury risk limits approved.
- Governance approval recorded.
- Treasury Safe approval recorded.
- Public disclosure updated if required.
- Incident response process current.
- Mainnet monitor passing.

## Confidentiality rule

Do not publish bank account details, private wallet-control details, sensitive entity details, private keys, seed phrases, signing procedures, or private compliance notes.

Public pages should only show sanitized status.

## Rule

No real treasury funding should occur until the source is approved and a separate funding transaction package is prepared.
