# DEX Public Trading Link Approval

## Purpose

Record approval for the public DEX trading link after public trading has been approved and DEX liquidity has been verified.

## Current status after this milestone

Liquidity Safe transaction executed: yes.

Liquidity added: yes.

Position minted on-chain: yes.

Public trading approved: yes.

Public trading link approval: recorded.

Buy page activated: no.

Full launch approved: no.

## What this approval can approve

- The approved public trading link target.
- Proceeding to a later buy page activation approval milestone.
- Proceeding to a later buy page activation/live milestone.

## What this approval does not do

- Activate the buy page.
- Approve global treasury funding.
- Approve full launch.

## Rule

Public trading link approval is not buy page activation and is not full launch approval.
