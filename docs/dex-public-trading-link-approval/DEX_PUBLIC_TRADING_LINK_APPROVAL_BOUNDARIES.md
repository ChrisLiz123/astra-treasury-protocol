# DEX Public Trading Link Approval Boundaries

## This package can do

- Record approval for a public DEX trading link.
- Confirm liquidity and position evidence.
- Confirm public trading approval already exists.
- Confirm buy page and full launch remain off.
- Publish approval status.

## This package cannot do

- Activate the buy page.
- Approve global treasury funding.
- Approve full launch.

## Rule

Approval only.
