# DEX Public Trading Link Approval Checklist

## Required before recording approval

- [x] DEX public trading approval recorded.
- [x] Public trading approved.
- [x] DEX liquidity post-execution verification complete.
- [x] Liquidity Safe transaction executed.
- [x] Liquidity added.
- [x] Position minted.
- [x] Pool liquidity greater than zero.
- [x] Position owner is liquidity Safe.
- [x] Buy page not activated.
- [x] Full launch not approved.

## Required after approval

- [ ] Buy page activation approval.
- [ ] Buy page activation live.
- [ ] Full launch approval only after separate approvals.

## Rule

Do not activate the buy page during this approval milestone.
