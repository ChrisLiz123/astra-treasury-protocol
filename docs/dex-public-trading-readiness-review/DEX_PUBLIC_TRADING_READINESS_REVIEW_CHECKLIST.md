# DEX Public Trading Readiness Review Checklist

## Required checks

- [x] Post-execution liquidity verification complete.
- [x] Liquidity Safe transaction executed.
- [x] Liquidity added.
- [x] Position minted.
- [x] Pool liquidity greater than zero.
- [x] Position owner verified.
- [x] Position details match payload.
- [x] Public trading not approved.
- [x] Public trading live status not present.
- [x] Buy page not activated.
- [x] Full launch not approved.

## Required after readiness review

- [ ] Public trading approval.
- [ ] Public trading publication/link approval.
- [ ] Buy page activation approval.
- [ ] Full launch approval only after separate approvals.

## Rule

Do not enable public trading during this milestone.
