# DEX Parameter Approval Boundaries

## Approved by this package

Selected DEX/liquidity parameters only.

## Not approved by this package

- Creating a Uniswap pool.
- Adding ASTRA liquidity.
- Adding counter-asset liquidity.
- Publishing a public trading link.
- Activating a buy page.
- Generating a Safe payload.
- Executing a Safe transaction.
- Moving treasury assets.
- Enabling public trading.
- Approving full launch.

## Rule

Later action-specific approval is still required before any on-chain or public trading action.
