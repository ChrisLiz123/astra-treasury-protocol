# Governance Vote Authorization Blockers

The following block vote-opening authorization:

- Vote title not reviewed.
- Vote summary not reviewed.
- Vote scope not reviewed.
- Resolution text not reviewed.
- Capability matrix not reviewed.
- Public notice not reviewed.
- Public status update not reviewed.
- Vote timeline not reviewed.
- Evidence plan not reviewed.
- Voter or signer list not confirmed.
- Monitoring status not reviewed.
- Incident status not reviewed.
- Final vote authorization not recorded.

## Rule

Governance vote authorization preparation is not vote authorization.
