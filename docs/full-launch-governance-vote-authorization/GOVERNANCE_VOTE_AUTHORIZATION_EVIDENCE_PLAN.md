# Governance Vote Authorization Evidence Plan

## Purpose

Define what evidence must be preserved before a governance vote opening is authorized.

## Required evidence

- Pre-authorization public status snapshot.
- Vote scope document.
- Resolution text.
- Capability activation matrix.
- Governance vote opening checklist.
- Public notice draft.
- Public status update draft.
- Voter or signer list confirmation.
- Monitoring status.
- Incident status.
- Authorization decision record.
- Post-authorization evidence snapshot, if authorization is later recorded.

## Rule

Vote opening authorization should not be treated as valid without evidence.
