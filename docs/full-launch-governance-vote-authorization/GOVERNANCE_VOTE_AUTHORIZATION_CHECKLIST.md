# Governance Vote Authorization Checklist

## Required before authorizing vote opening

- [ ] Vote title reviewed.
- [ ] Vote summary reviewed.
- [ ] Vote scope reviewed.
- [ ] Resolution text reviewed.
- [ ] Capability matrix reviewed.
- [ ] Public notice reviewed.
- [ ] Public status update reviewed.
- [ ] Vote timeline reviewed.
- [ ] Evidence plan reviewed.
- [ ] Voter or signer list confirmed.
- [ ] Monitoring status reviewed.
- [ ] Incident status reviewed.
- [ ] Final vote authorization recorded.

## Rule

Do not authorize a governance vote opening until every checklist item is complete.
