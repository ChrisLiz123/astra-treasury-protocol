# Governance Vote Authorization Public Notice

## Current public notice

AstraTreasury has prepared a governance vote authorization package.

No vote opening is authorized. No governance vote is open. No governance decision is recorded. Full launch and restricted capabilities are not approved.

## If vote authorization is later recorded

The public notice must include:

- Authorization date.
- Vote title.
- Vote scope.
- Vote opening conditions.
- Capabilities under consideration.
- Capabilities that remain disabled.
- Evidence reference.
- Public status links.

## Rule

Public language must not imply a vote is open until vote opening is actually authorized and published.
