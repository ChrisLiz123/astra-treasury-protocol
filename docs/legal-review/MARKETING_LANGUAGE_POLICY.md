# Marketing Language Policy

## Purpose

Prevent public language that could misrepresent AstraTreasury or imply investment guarantees.

## Prohibited language

- Guaranteed returns
- Passive income
- Risk-free treasury
- AI profit machine
- Guaranteed buybacks
- Guaranteed price support
- Self-pumping token
- Holders will get rich
- Treasury guarantees appreciation

## Safer language

- Testnet prototype
- AI-assisted treasury governance
- Human-approved signal workflow
- Policy-gated execution
- Transparent on-chain signal logging
- Base Sepolia research prototype

## Required disclaimer

AstraTreasury v0.1.1 is a Base Sepolia testnet prototype. It is not an investment product, not a public token sale, and not a promise of returns. Testnet ASTP has no real monetary value.
