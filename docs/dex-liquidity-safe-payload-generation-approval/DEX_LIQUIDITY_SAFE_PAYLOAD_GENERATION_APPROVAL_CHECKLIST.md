# DEX Liquidity Safe Payload Generation Approval Checklist

## Required before recording approval

- [x] Liquidity mint calldata generated.
- [x] Liquidity mint calldata verified.
- [x] Mint calldata hash verified.
- [x] Live token balances verified.
- [x] Live token allowances verified.
- [x] Pool liquidity remains zero.
- [x] Liquidity Safe payload not generated.
- [x] Liquidity Safe transaction not submitted.
- [x] Liquidity Safe transaction not executed.
- [x] Liquidity not added.
- [x] Position not minted.
- [x] Public trading not approved.
- [x] Full launch not approved.

## Required after approval

- [ ] Liquidity Safe payload generation.
- [ ] Liquidity Safe payload verification.
- [ ] Liquidity Safe submission approval.
- [ ] Liquidity Safe submission dry run.

## Rule

Do not generate the Safe payload or add liquidity during this approval milestone.
