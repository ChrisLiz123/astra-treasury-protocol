# DEX Liquidity Safe Payload Generation Approval Boundaries

## This package can do

- Record approval for later liquidity Safe payload generation.
- Confirm verified mint calldata exists.
- Confirm no liquidity has been added.
- Publish approval status.

## This package cannot do

- Generate a liquidity Safe payload.
- Submit a liquidity Safe transaction.
- Execute a liquidity Safe transaction.
- Add liquidity.
- Mint a position on-chain.
- Enable public trading.
- Activate the buy page.

## Rule

Approval-only.
