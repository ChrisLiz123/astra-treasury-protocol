# Before Touching Production

## Required checks

- [ ] Confirm current mode is restricted-mainnet-operation.
- [ ] Confirm governance decision is recorded.
- [ ] Confirm full launch is not approved.
- [ ] Confirm public token sale is not approved.
- [ ] Confirm treasury funding is not approved.
- [ ] Confirm Safe payload is not generated.
- [ ] Confirm Safe transaction is not prepared.
- [ ] Confirm mainnet execution queue is disabled.
- [ ] Confirm mainnet monitor is passing.
- [ ] Confirm alerts do not require response.
- [ ] Confirm active incidents are zero.
- [ ] Confirm intended action is maintenance, not capability activation.

## Rule

If any check fails, stop and open or update an incident record.
