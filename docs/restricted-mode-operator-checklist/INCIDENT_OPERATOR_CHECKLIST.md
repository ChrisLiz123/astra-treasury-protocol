# Incident Operator Checklist

## If an incident is detected

- [ ] Open or update the incident record.
- [ ] Preserve evidence.
- [ ] Review monitor output.
- [ ] Review alerts.
- [ ] Confirm whether public status must be updated.
- [ ] Do not activate any capability.
- [ ] Do not generate a Safe payload.
- [ ] Do not enable execution queue.
- [ ] Run evidence archive after updates.
- [ ] Document resolution or follow-up.

## Rule

Incident response does not authorize launch, funding, execution, or capability activation.
