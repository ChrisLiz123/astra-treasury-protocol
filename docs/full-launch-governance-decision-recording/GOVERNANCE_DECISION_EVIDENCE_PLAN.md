# Governance Decision Evidence Plan

## Purpose

Define evidence required before a governance decision can be recorded.

## Required evidence before recording

- Vote authorization record.
- Vote opening record.
- Vote result.
- Signed resolution.
- Resolution signing authorization.
- Capability activation matrix.
- Public status update draft.
- Monitoring status.
- Incident status.
- Evidence archive snapshot.

## Required evidence after recording, if later recorded

- Final governance decision record.
- Public status update.
- Updated full-launch status.
- Updated capability status.
- Evidence archive snapshot.

## Rule

No governance decision should be treated as recorded without evidence.
