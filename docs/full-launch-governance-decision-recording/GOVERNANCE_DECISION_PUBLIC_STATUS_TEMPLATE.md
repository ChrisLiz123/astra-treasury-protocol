# Governance Decision Public Status Template

## If decision is not recorded

Governance decision recording package is prepared for review. No governance decision is recorded. Full launch and restricted capabilities are not approved.

## If a decision is later recorded

A public status update must include:

- decision date
- vote result
- resolution reference
- exact approved capabilities
- exact disabled capabilities
- evidence reference
- next activation milestone

## Rule

Public language must not imply approval before the governance decision is recorded.
