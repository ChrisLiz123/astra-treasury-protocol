# Governance Resolution Authorization Evidence Plan

## Purpose

Define evidence required before resolution signing can be authorized.

## Required evidence before authorization

- Vote authorization record.
- Vote opening record.
- Vote URL.
- Vote result.
- Vote result evidence.
- Final resolution text.
- Capability activation matrix.
- Signer or authority list.
- Monitoring status.
- Incident status.
- Public status update draft.
- Evidence archive snapshot.

## Required evidence after authorization, if later recorded

- Authorization decision record.
- Authorization timestamp.
- Authorizer evidence.
- Updated public status.
- Evidence archive snapshot.

## Rule

Resolution signing authorization should not be treated as valid without evidence.
