# Governance Resolution Authorization Public Status Template

## If authorization is not recorded

Governance resolution signing authorization package is prepared for review. Resolution signing is not authorized. No resolution is signed. No governance decision is recorded. Full launch and restricted capabilities are not approved.

## If authorization is later recorded

A public status update must include:

- authorization date
- authorizer
- resolution reference
- vote result reference
- exact capabilities under consideration
- exact capabilities still disabled
- evidence reference
- next signing milestone

## Rule

Public language must not imply approval before a signed governance resolution exists.
