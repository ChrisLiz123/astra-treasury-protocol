# Capability Request Review Checklist

## Required for review readiness

- [ ] Request file is either absent or valid.
- [ ] Requested capability is allowed.
- [ ] Requester reference is present.
- [ ] Authority reference is present.
- [ ] Purpose is present.
- [ ] Scope is present.
- [ ] Evidence reference is present.
- [ ] Public status plan is present.
- [ ] Request does not request immediate Safe payload generation.
- [ ] Request does not request immediate execution queue activation.
- [ ] Request does not request immediate funds movement.
- [ ] Restricted-mode operator checklist is ready.
- [ ] Capability intake gate is healthy.
- [ ] Mainnet monitor is passing.
- [ ] Active incidents are zero.
- [ ] Execution queue remains disabled.

## Rule

Review readiness is not approval.
