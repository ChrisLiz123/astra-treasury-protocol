# Capability Request Review Runbook

## Step 1

Confirm request import gate passes.

## Step 2

Run capability intake gate.

## Step 3

Run capability request review gate.

## Step 4

If no active request exists, remain in standby.

## Step 5

If a valid request exists, open a dedicated action-specific path for that exact capability.

## Step 6

Do not update capability approval flags until the dedicated path is approved.

## Rule

The review gate does not approve the request.
