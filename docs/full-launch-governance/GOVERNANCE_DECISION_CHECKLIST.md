# Governance Decision Checklist

## Review areas

- [ ] Full-launch scope reviewed.
- [ ] Capability approvals reviewed.
- [ ] Treasury risk limits reviewed.
- [ ] Funding source reviewed.
- [ ] Treasury Safe approval process reviewed.
- [ ] Treasury transaction dry run reviewed.
- [ ] Treasury disclosure approval track reviewed.
- [ ] Public disclosures reviewed.
- [ ] Monitoring status reviewed.
- [ ] Alert and incident workflow reviewed.
- [ ] Operator runbook reviewed.
- [ ] Public status update reviewed.
- [ ] Final governance signoff recorded.

## Decision requirements

- [ ] Governance approver assigned.
- [ ] Governance vote or resolution completed.
- [ ] Final decision record signed.
- [ ] Evidence snapshot archived.
- [ ] Public status updated.

## Rule

Do not mark full launch approved until every checklist item is complete.
