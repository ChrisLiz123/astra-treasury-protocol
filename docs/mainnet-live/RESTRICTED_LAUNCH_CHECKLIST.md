# Mainnet Restricted Launch Checklist v1

## Status

AstraTreasury Base Mainnet contracts are deployed in restricted operational mode.

## Required passing checks

- [ ] Mainnet post-deployment verification passing.
- [ ] Mainnet operational monitor passing.
- [ ] Mainnet event monitor has no high/critical alerts.
- [ ] Alert escalation status requires no active response.
- [ ] Incident summary has no active incidents.
- [ ] Restricted operations validator passing.
- [ ] Mainnet execution queue remains disabled.
- [ ] Public pages are reachable.
- [ ] Operator notification workflow is active or intentionally disabled.

## Still not approved

- Public token sale.
- Real treasury funding.
- Staking or rewards.
- Buyback program.
- Autonomous execution.
- Mainnet execution queue activation.
- Paper-to-on-chain automation.

## Rule

Restricted launch status is not the same as unrestricted launch.
