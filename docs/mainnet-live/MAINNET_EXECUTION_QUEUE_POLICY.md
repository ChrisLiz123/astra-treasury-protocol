# Mainnet Execution Queue Policy

## Status

Mainnet execution queue: disabled.

## Current mode

AstraTreasury is in restricted Base Mainnet operation.

Contracts are deployed and monitored, but mainnet treasury execution is not active.

## Disabled capabilities

- Mainnet execution queue
- Paper-to-on-chain automation
- Autonomous execution
- Real treasury funding
- Public token sale
- Staking or rewards
- Buyback program

## Activation requirements

Mainnet execution queue activation requires a separate milestone and explicit approvals:

- Governance approval
- Legal review update
- Treasury funding approval
- Incident response drill
- Mainnet execution dry run
- Public disclosure update

## Rule

No mainnet treasury execution should occur while this queue remains disabled.
