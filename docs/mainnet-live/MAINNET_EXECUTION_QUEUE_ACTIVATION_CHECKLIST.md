# Mainnet Execution Queue Activation Checklist

## Purpose

Define what must happen before any mainnet execution queue is enabled.

## Required approvals

- [ ] Governance approval recorded.
- [ ] Legal review updated.
- [ ] Treasury funding approval recorded.
- [ ] Incident response drill completed.
- [ ] Mainnet execution dry run completed.
- [ ] Public restricted-operations page updated.
- [ ] Public monitor page updated.
- [ ] Safe owners available.
- [ ] Executor Safe process reviewed.
- [ ] Treasury risk limits reviewed.

## Technical requirements

- [ ] ExecutionController role state verified.
- [ ] TreasuryPolicy limits reviewed.
- [ ] Mainnet monitor passing.
- [ ] Restricted operations validator passing.
- [ ] Queue dry-run status passing.
- [ ] No autonomous execution enabled.

## Rule

Activation is blocked until every item is complete.
