# DEX Pool Creation Readiness Runbook

## Step 1

Run DEX parameter approval gate.

## Step 2

Run DEX liquidity source/Safe-impact gate.

## Step 3

Run DEX pool creation readiness gate.

## Step 4

If ready, proceed to an on-chain pool existence/factory precheck.

## Step 5

Do not create a pool or generate a Safe payload from this gate.

## Rule

The next pool-related milestone must remain approval/precheck until a dedicated execution step is approved.
