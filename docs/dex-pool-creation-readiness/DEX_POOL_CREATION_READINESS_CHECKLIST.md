# DEX Pool Creation Readiness Checklist

## Required readiness checks

- [x] DEX/liquidity path selected.
- [x] DEX parameters approved.
- [x] Liquidity source / Safe-impact planning approved.
- [x] Valid parameter selection exists.
- [x] Restricted-mode final release ready.
- [x] Governance decision recorded.
- [x] Capability Matrix remains all-disabled.
- [x] Mainnet monitor passing.
- [x] Active incidents zero.
- [x] Execution queue disabled.
- [x] Treasury funding not approved.
- [x] Safe payload not generated.
- [x] Full launch not approved.

## Required before later pool-creation approval

- [ ] On-chain pool existence check completed.
- [ ] DEX factory/router reviewed.
- [ ] Token approval requirements reviewed.
- [ ] Safe payload plan reviewed if needed.
- [ ] Public risk disclosure approved.
- [ ] Pool-creation action approval recorded.

## Rule

Readiness is not execution.
