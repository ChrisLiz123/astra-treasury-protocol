# DEX Pool Creation Boundaries

## This gate can do

- Confirm prerequisites for a later pool-creation approval package.
- Confirm selected parameters and source/Safe-impact planning are present.
- Confirm restricted-mode controls remain intact.

## This gate cannot do

- Create a pool.
- Add liquidity.
- Generate a Safe payload.
- Execute a Safe transaction.
- Move funds.
- Publish a trading link.
- Activate a buy page.
- Approve public trading.
- Approve full launch.

## Rule

A separate pool-creation approval package is required before any on-chain pool action.
