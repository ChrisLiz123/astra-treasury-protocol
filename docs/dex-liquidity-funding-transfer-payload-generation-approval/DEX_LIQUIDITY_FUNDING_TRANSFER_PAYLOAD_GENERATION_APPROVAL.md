## What this approval can approve

- Proceeding to a later payload-generation step.
- Using the approved source address and destination Safe.
- Using the approved token shortfalls and transfer amounts.
- Generating a reviewable funding-transfer Safe payload in the next milestone.

## What this approval does not approve

- Payload generation during this approval step.
- Safe submission.
- Safe execution.
- Treasury funds movement.
- Token approval execution.
- Liquidity addition.
- Position minting.
- Public trading.
- Buy page activation.
- Full launch.

## Rule

Payload-generation approval is not payload generation and is not funds movement.
