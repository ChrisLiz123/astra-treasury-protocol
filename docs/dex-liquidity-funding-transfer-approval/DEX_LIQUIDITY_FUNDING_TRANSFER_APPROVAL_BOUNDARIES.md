# DEX Liquidity Funding Transfer Approval Boundaries

## This package can do

- Record approval of the calculated funding shortfalls.
- Record approval of the source and destination.
- Confirm no funding payload or transfer has occurred.
- Preserve public evidence.

## This package cannot do

- Generate transfer calldata.
- Generate Safe payloads.
- Submit a Safe transaction.
- Execute a transfer.
- Execute token approvals.
- Add liquidity.
- Mint a position.
- Enable public trading.
- Activate the buy page.

## Rule

Approval-only.
