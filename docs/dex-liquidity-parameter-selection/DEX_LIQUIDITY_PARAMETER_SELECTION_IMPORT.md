## Importable selection fields

- DEX venue.
- Pool version.
- Token pair.
- ASTRA token address.
- Counter-asset symbol and address.
- Fee tier or pool type.
- Initial price approach.
- Initial price.
- Liquidity amount.
- Liquidity source.
- Price range.
- Slippage guidance.
- Impermanent-loss disclosure.
- MEV/front-running disclosure.
- Token-impersonation disclosure.
- Safe transaction path.
- Public trading link plan.
- Buy page language.

## Rule

Imported parameter selection is not approval and is not pool creation.
