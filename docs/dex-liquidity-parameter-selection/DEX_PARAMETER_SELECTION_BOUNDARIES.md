# DEX Parameter Selection Boundaries

## Import can do

- Record selected parameters for review.
- Show whether a selection exists.
- Prepare for later finalization review.

## Import cannot do

- Finalize the selected parameters.
- Create a pool.
- Add liquidity.
- Generate a Safe payload.
- Execute a Safe transaction.
- Move treasury assets.
- Activate a trading link.
- Activate the buy page.
- Approve full launch.

## Rule

Selection import must stay review-only.
