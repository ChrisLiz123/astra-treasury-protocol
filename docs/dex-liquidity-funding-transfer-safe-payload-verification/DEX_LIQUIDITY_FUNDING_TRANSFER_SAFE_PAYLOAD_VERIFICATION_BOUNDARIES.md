# DEX Liquidity Funding Transfer Safe Payload Verification Boundaries

## This package can do

- Verify the generated payload.
- Verify calldata and transfer amounts.
- Verify source balances.
- Verify no funds have moved.
- Publish verification evidence.

## This package cannot do

- Submit the Safe transaction.
- Execute the Safe transaction.
- Move funds.
- Execute token approvals.
- Add liquidity.
- Mint a position.
- Enable public trading.
- Activate the buy page.

## Rule

Read-only verification.
