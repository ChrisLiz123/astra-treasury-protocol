# Capability Activation Intake Runbook

## Step 1

Receive the future capability activation request.

## Step 2

Confirm the request names exactly one capability.

## Step 3

Confirm all restricted-mode gates still pass.

## Step 4

Import the sanitized request file only if complete.

## Step 5

Run the intake gate.

## Step 6

If accepted for review, create a separate action-specific approval path.

## Rule

Do not generate Safe payloads, move funds, enable execution, or update approvals from intake alone.
