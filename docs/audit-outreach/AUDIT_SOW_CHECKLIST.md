# Audit Statement of Work Checklist

## Required SOW items

- Project name
- Version under review
- Git commit hash
- Contract scope
- Off-chain workflow scope
- Out-of-scope items
- Audit start date
- Audit end date
- Retest policy
- Report delivery format
- Public report permission
- Fee or pricing model
- Payment schedule
- Communication channel
- Disclosure policy
- Confidentiality terms

## Scope must include

- AstraToken
- TreasuryPolicy
- TreasuryVault
- SignalRegistry
- ExecutionController
- Signal approval workflow
- Policy-aware execution workflow
- Cancelled-signal regression
- Role-transfer planning
- Mainnet readiness notes

## Scope should explicitly exclude

- Mainnet deployment authorization
- Public token sale review unless legal counsel adds it
- Investment product design
- Exchange listing review
