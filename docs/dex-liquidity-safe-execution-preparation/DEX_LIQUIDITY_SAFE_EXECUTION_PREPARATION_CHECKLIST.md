# DEX Liquidity Safe Execution Preparation Checklist

## Required checks

- [x] Liquidity Safe execution approval recorded.
- [x] Pending signatures threshold reached.
- [x] Missing confirmation count is zero.
- [x] Safe tx hash recorded.
- [x] Safe nonce recorded.
- [x] Safe transaction still pending/not executed.
- [x] Safe payload hash verified.
- [x] Transaction Builder hash verified.
- [x] Live balances and allowances verified.
- [x] Pool liquidity remains zero.
- [x] Liquidity not added.
- [x] Position not minted.
- [x] Public trading not approved.
- [x] Full launch not approved.

## Required after preparation

- [ ] Liquidity Safe execution live.
- [ ] Post-execution liquidity verification.
- [ ] Position mint verification.

## Rule

Do not execute during this preparation milestone.
