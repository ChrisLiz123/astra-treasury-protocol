# DEX Liquidity Safe Execution Preparation Boundaries

## This package can do

- Prepare Safe execution instructions.
- Confirm threshold remains reached.
- Confirm the Safe transaction is still pending/not executed.
- Publish execution preparation status.

## This package cannot do

- Execute the Safe transaction.
- Add liquidity.
- Mint a position on-chain.
- Enable public trading.
- Activate the buy page.
- Approve full launch.

## Rule

Preparation only.
