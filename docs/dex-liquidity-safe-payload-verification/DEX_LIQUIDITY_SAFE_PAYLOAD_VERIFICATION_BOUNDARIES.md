# DEX Liquidity Safe Payload Verification Boundaries

## This package can do

- Verify the generated Safe payload.
- Verify the Transaction Builder JSON.
- Verify live balances and allowances.
- Verify pool liquidity remains zero.
- Publish verification evidence.

## This package cannot do

- Submit a liquidity Safe transaction.
- Execute a liquidity Safe transaction.
- Add liquidity.
- Mint a position on-chain.
- Enable public trading.
- Activate the buy page.

## Rule

Verification only.
