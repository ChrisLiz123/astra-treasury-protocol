# Governance Decision Recording Live Blockers

The following block governance decision recording:

- Live precheck not ready.
- Action-specific approval not recorded.
- Signed resolution evidence missing or invalid.
- Capability Matrix not finalized.
- Public Status Update not finalized.
- Active incidents present.
- Mainnet monitor not passing.
- Mainnet execution queue enabled.
- Treasury funding approved unexpectedly.
- Safe payload generated unexpectedly.
- Any capability approved unexpectedly.

## Rule

Any blocker prevents governance decision recording.
