# Governance Decision Recording Live Package

## Purpose

Record the restricted-mode / all-disabled governance decision after the live precheck is ready.

## Current decision scope

Restricted Base Mainnet operation.

## Approved capabilities

None.

## Disabled capabilities

- Public token sale.
- Real treasury funding.
- Staking or rewards.
- Buyback program.
- Mainnet execution queue.
- Paper-to-on-chain automation.
- Autonomous execution.
- Safe transaction payload generation.
- Safe transaction execution.

## Required before recording

- Governance decision live precheck ready.
- Action-specific approval recorded.
- Signed resolution evidence imported and validated.
- Capability Matrix finalized.
- Public Status Update finalized.
- Mainnet monitor passing.
- No active incidents.
- Mainnet execution queue disabled.
- Treasury funding not approved.
- Safe payload not generated.

## Not approved by this package

- Full launch.
- Public token sale.
- Real treasury funding.
- Staking or rewards.
- Buyback program.
- Execution queue activation.
- Paper-to-on-chain automation.
- Autonomous execution.
- Safe payload generation.
- Safe transaction execution.

## Rule

This package records only the restricted-mode / all-disabled governance decision.
