# Signed Resolution Evidence Import Blockers

The following block signed-resolution evidence from satisfying the governance decision path:

- Evidence file missing.
- Evidence file invalid JSON.
- Resolution title missing.
- Resolution reference missing.
- Resolution hash missing.
- Signed timestamp missing.
- Governance resolution signed flag not true.
- Resolution signing authorization recorded flag not true.
- Resolution signing authorization reference missing.
- Vote result reference missing.
- Capability matrix reference missing.
- Public status update reference missing.
- Evidence reference missing.
- Approved capabilities not empty for the current all-disabled path.
- Private or sensitive data included.

## Rule

The governance decision path remains blocked until valid signed-resolution evidence is imported and validated.
