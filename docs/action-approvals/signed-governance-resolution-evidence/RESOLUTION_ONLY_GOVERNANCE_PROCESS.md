# Resolution-Only Governance Process

## Governance mode

Resolution-only.

## Meaning

A separate vote/result evidence file is not required.

The signed governance resolution evidence is the controlling governance evidence for the governance decision path.

## Required evidence

- Governance process mode: resolution-only.
- Governance process reference.
- Resolution title.
- Resolution reference.
- Resolution hash.
- Signed timestamp.
- Resolution signing authorization reference.
- Capability matrix reference.
- Public status update reference.
- Evidence reference.
- Approved capabilities must be empty for the current all-disabled restricted-mode path.

## Rule

Do not create fake vote evidence. Use the signed governance resolution evidence as the controlling governance evidence.
