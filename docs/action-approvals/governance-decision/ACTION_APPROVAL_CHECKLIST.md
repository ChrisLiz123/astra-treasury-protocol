# Action Approval Checklist: Governance Decision Recording

## Required before approval

- [ ] Selected action reviewed.
- [ ] Scope reviewed.
- [ ] Vote result reviewed.
- [ ] Signed resolution reviewed.
- [ ] Resolution signing authorization reviewed.
- [ ] Capability matrix reviewed.
- [ ] Approved capabilities reviewed.
- [ ] Disabled capabilities reviewed.
- [ ] Public status update reviewed.
- [ ] Evidence reference reviewed.
- [ ] Decision recorder assigned.
- [ ] Monitoring status reviewed.
- [ ] Incident status reviewed.
- [ ] Final action approval recorded.

## Rule

Do not record the governance decision until every checklist item is complete.
