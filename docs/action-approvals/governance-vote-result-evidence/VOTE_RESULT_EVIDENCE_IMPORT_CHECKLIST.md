# Vote/Result Evidence Import Checklist

## Before importing evidence

- [ ] Evidence is real.
- [ ] Evidence is sanitized.
- [ ] Vote title is present.
- [ ] Vote URL is present.
- [ ] Vote mechanism is present.
- [ ] Vote opened timestamp is present.
- [ ] Vote closed timestamp is present.
- [ ] Vote completed status is present.
- [ ] Vote result is present.
- [ ] Evidence reference is present.
- [ ] Capability matrix reference is present.
- [ ] Public status update reference is present.
- [ ] No private voter data is included.
- [ ] No private signer data is included.
- [ ] No keys, seed phrases, or secrets are included.

## Rule

Do not import evidence unless every item is complete.
