# Vote/Result Evidence Import Blockers

The following block vote/result evidence from satisfying the governance decision path:

- Evidence file missing.
- Evidence file invalid JSON.
- Vote URL missing.
- Vote mechanism missing.
- Vote opened timestamp missing.
- Vote closed timestamp missing.
- Vote opened flag not true.
- Vote completed flag not true.
- Vote result recorded flag not true.
- Result missing.
- Evidence reference missing.
- Capability matrix reference missing.
- Public status update reference missing.
- Private or sensitive data included.

## Rule

The governance decision path remains blocked until valid evidence is imported and validated.
