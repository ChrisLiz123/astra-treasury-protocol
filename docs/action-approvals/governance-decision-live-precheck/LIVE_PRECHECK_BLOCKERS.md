# Governance Decision Live Precheck Blockers

The following block live governance decision recording:

- Vote/result evidence missing or invalid.
- Signed resolution evidence missing or invalid.
- Action-specific approval not recorded.
- Decision recorder not assigned.
- Mainnet monitor not passing.
- Active incidents present.
- Capability matrix not finalized.
- Public status update not finalized.
- Safe payload already generated unexpectedly.
- Treasury funding approved unexpectedly.
- Mainnet execution queue enabled unexpectedly.

## Rule

Any blocker keeps governance decision recording disabled.
