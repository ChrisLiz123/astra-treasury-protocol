# Incident Response Handoff

## Current incident posture

Active incidents should remain zero.

## If an incident is detected

- Record the incident.
- Preserve evidence.
- Review monitor and alert outputs.
- Do not activate any capability.
- Update public status if user-facing or material.
- Archive evidence after resolution.

## Rule

Incident response is operational safety work. It does not authorize launch, funding, execution, or capability activation.
