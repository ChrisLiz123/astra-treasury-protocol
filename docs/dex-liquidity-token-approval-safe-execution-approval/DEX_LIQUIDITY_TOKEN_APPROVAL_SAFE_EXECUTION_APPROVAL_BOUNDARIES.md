# DEX Liquidity Token Approval Safe Execution Approval Boundaries

## This package can do

- Record approval for later Safe execution.
- Confirm Safe threshold is reached.
- Confirm the Safe transaction is still not executed.
- Confirm token approvals have not executed.
- Publish execution approval evidence.

## This package cannot do

- Execute the Safe transaction.
- Execute token approvals.
- Generate liquidity calldata.
- Add liquidity.
- Mint a position.
- Enable public trading.
- Activate the buy page.

## Rule

Approval-only.
