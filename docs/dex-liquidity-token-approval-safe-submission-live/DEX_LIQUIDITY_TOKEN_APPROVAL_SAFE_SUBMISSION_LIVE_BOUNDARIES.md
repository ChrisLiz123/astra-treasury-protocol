# DEX Liquidity Token Approval Safe Submission Live Boundaries

## This package can do

- Record Safe submission/proposal evidence.
- Confirm Safe Transaction Service shows not executed.
- Publish submission status.

## This package cannot do

- Execute token approvals.
- Generate liquidity calldata.
- Add liquidity.
- Mint a position.
- Enable public trading.
- Activate the buy page.
- Approve full launch.

## Rule

Submission evidence only.
