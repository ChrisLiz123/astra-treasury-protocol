# Daily Maintenance Checklist

- [ ] Review mainnet monitor status.
- [ ] Confirm alerts do not require response.
- [ ] Confirm active incidents are zero.
- [ ] Confirm execution queue remains disabled.
- [ ] Confirm treasury funding remains not approved.
- [ ] Confirm Safe payload remains not generated.
- [ ] Confirm public site is reachable.

## Rule

If any check fails, open or update an incident record before any other action.
