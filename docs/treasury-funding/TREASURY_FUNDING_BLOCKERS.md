# Treasury Funding Blockers

The following block treasury funding:

- Legal review incomplete.
- Accounting review incomplete.
- Governance approval missing.
- Treasury Safe approval missing.
- Source of funds not documented.
- Risk limits not approved.
- Public disclosures not updated.
- Incident response drill not current.
- Monitoring status not confirmed.
- Funding transaction package not created.

## Rule

Restricted launch stabilization is not treasury funding approval.
