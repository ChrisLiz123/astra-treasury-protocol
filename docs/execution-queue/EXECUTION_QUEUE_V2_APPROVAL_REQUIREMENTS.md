# Execution Queue v2 Approval Requirements

## Required before any mainnet execution queue activation

- Legal full-launch review complete.
- Governance approval recorded.
- Treasury funding approval recorded, if funds are involved.
- Treasury risk limits approved.
- Mainnet execution dry run completed.
- Incident response drill updated.
- Public disclosure updated.
- Operator runbook updated.
- Mainnet monitor passing.
- Alert and incident workflow active.

## Required controls

- Governance Safe approval.
- Executor Safe review.
- Manual execution path.
- No autonomous execution for v1.
- Public status update before activation.

## Rule

No execution queue activation is approved by this dry run.
