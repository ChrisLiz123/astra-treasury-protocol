# DEX Liquidity Token Approval Safe Submission Preparation Boundaries

## This package can do

- Prepare Safe submission instructions.
- Prepare Transaction Builder import reference.
- Recheck that approvals have not executed.
- Publish preparation evidence.

## This package cannot do

- Submit the Safe transaction.
- Execute token approvals.
- Generate liquidity calldata.
- Add liquidity.
- Mint a position.
- Enable public trading.
- Activate the buy page.

## Rule

Preparation only.
