# Full Launch Approval Checklist

## Required before recording approval

- [x] Full launch readiness review complete.
- [x] Buy page activated.
- [x] Public trading live.
- [x] Public trading approved.
- [x] Public trading link approved.
- [x] DEX liquidity post-execution verification complete.
- [x] Liquidity added.
- [x] Position minted.
- [x] Pool liquidity greater than zero.
- [x] Position owner is liquidity Safe.
- [x] Global treasury funding not approved.
- [x] Global treasury funding not executed.
- [x] Full launch not previously approved.

## Required after approval

- [ ] Full launch live/public finalization.
- [ ] Global treasury funding only after separate approval.

## Rule

Do not approve or execute global treasury funding during this milestone.
