# DEX Liquidity Token Approval Payload Generation Approval Boundaries

## This package can do

- Record approval for a later token-approval payload-generation step.
- Carry forward rechecked token approval requirements.
- Confirm no token approval has executed.
- Publish approval status.

## This package cannot do

- Generate token approval Safe payloads.
- Execute token approvals.
- Generate liquidity calldata.
- Add liquidity.
- Mint a position.
- Enable public trading.
- Activate the buy page.

## Rule

Approval-only.
