# Governance Vote Opening Execution Blockers

The following block vote opening:

- Governance vote authorization not recorded.
- Vote scope not final-approved.
- Resolution text not final-approved.
- Capability matrix not final-approved.
- Public vote notice not final-approved.
- Voter or signer list not confirmed.
- Evidence plan not final-approved.
- Opening operator not assigned.
- Monitoring status not reviewed.
- Incident status not reviewed.

## Rule

Vote opening execution preparation is not vote opening.
