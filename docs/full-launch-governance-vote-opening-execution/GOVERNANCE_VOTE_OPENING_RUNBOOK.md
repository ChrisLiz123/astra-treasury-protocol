# Governance Vote Opening Runbook

## Before opening a vote

1. Confirm vote authorization is recorded.
2. Confirm vote scope is final-approved.
3. Confirm resolution text is final-approved.
4. Confirm capability matrix is final-approved.
5. Confirm public notice is final-approved.
6. Confirm vote mechanism and timeline.
7. Confirm voter or signer list.
8. Capture pre-opening evidence snapshot.
9. Publish public vote notice.
10. Open vote.
11. Capture post-opening evidence snapshot.
12. Update public status board.

## Current status

This runbook is drafted only. No vote opening is authorized.

## Rule

Do not open a vote unless authorization is recorded and every prerequisite is complete.
