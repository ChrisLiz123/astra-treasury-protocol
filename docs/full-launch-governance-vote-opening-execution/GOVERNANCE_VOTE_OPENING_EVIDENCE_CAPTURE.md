# Governance Vote Opening Evidence Capture

## Purpose

Define evidence that must be captured if a governance vote is opened.

## Required evidence before opening

- Current Trust Center snapshot.
- Full-launch readiness snapshot.
- Governance vote authorization snapshot.
- Vote scope document.
- Resolution text.
- Capability matrix.
- Public notice draft.
- Monitoring status.
- Incident status.

## Required evidence after opening

- Vote URL.
- Open timestamp.
- Public notice URL.
- Post-opening public status snapshot.
- Evidence archive snapshot.

## Rule

No governance vote should be treated as open without evidence capture.
