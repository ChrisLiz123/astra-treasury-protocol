# Restricted-Mode Monitoring Baseline Checklist

## Required checks

- [x] Governance decision recorded.
- [x] Restricted-mode final status synced.
- [x] Capability Matrix finalized all-disabled.
- [x] Public Status Update finalized.
- [x] Mainnet monitor passing.
- [x] Alerts do not require response.
- [x] Active incidents zero.
- [x] Mainnet execution queue disabled.
- [x] Treasury funding not approved.
- [x] Treasury funding not executed.
- [x] Safe payload not generated.
- [x] Safe transaction not prepared.

## Rule

Any failed check blocks the monitoring baseline.
