# Announcement Publication Checklist

## Purpose

Prepare and track publication of the restricted Base Mainnet deployment announcement.

## Before posting

- [ ] Announcement page reviewed.
- [ ] Social post template reviewed.
- [ ] Long-form template reviewed.
- [ ] Public FAQ reviewed.
- [ ] Legal/marketing language reviewed.
- [ ] No public token sale language.
- [ ] No investment product language.
- [ ] No guaranteed return language.
- [ ] No staking/rewards launch language.
- [ ] No buyback program language.
- [ ] No autonomous execution language.
- [ ] Public links checked.
- [ ] Mainnet monitor passing.
- [ ] Alert status checked.
- [ ] Incident summary checked.
- [ ] Restricted launch status checked.

## Approved public wording

Use:

AstraTreasury Base Mainnet contracts are deployed in restricted operational mode.

Do not imply:

- Public token sale.
- Investment product.
- Real treasury funding.
- Staking or rewards.
- Buyback program.
- Autonomous execution.
- Unrestricted launch.

## Rule

Do not publish until all review items are complete.
