# Long-Form Announcement Template

## AstraTreasury Protocol Restricted Mainnet Deployment

AstraTreasury Protocol has completed a restricted Base Mainnet deployment.

This means the core contracts are deployed and visible on Base Mainnet, post-deployment checks have passed, Safe-based governance controls are configured, and monitoring is active.

This does not mean AstraTreasury has launched a public token sale or started real treasury operations.

## What is live

- Base Mainnet contract deployment.
- Public mainnet manifest.
- BaseScan contract links.
- Governance Safe setup.
- Post-deployment verification.
- Read-only mainnet monitor.
- Event monitor.
- Alert escalation workflow.
- Incident acknowledgement workflow.
- Public transparency pages.

## What remains disabled

- Public token sale.
- Real treasury funding.
- Staking or rewards.
- Buyback program.
- Autonomous execution.
- Mainnet execution queue.
- Paper-to-on-chain automation.
