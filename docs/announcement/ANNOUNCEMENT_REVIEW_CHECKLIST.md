# Announcement Review Checklist

## Required before public posting

- [ ] Security/operator review completed.
- [ ] Legal/marketing language reviewed.
- [ ] Public links checked.
- [ ] Mainnet monitor passing.
- [ ] Event monitor checked.
- [ ] Alert status checked.
- [ ] Incident summary checked.
- [ ] Restricted launch status checked.
- [ ] No prohibited investment language.
- [ ] No public sale implication.
- [ ] No staking/rewards implication.
- [ ] No buyback implication.
- [ ] No autonomous execution implication.

## Prohibited language

- Guaranteed returns.
- Passive income.
- Risk-free.
- Investment opportunity.
- Public sale is live.
- Staking is live.
- Rewards are live.
- Guaranteed buybacks.
- Treasury is funded.
- Autonomous trading is live.

## Rule

Do not post an announcement unless every required review item is complete.
