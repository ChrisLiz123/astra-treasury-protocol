# DEX Liquidity Mint Calldata Boundaries

## This package can do

- Generate local Uniswap V3 mint calldata.
- Record calldata hash and parameters.
- Publish calldata generation status.

## This package cannot do

- Generate a liquidity Safe payload.
- Submit a liquidity Safe transaction.
- Execute a liquidity Safe transaction.
- Add liquidity.
- Mint a position on-chain.
- Enable public trading.
- Activate the buy page.

## Rule

Local calldata only.
