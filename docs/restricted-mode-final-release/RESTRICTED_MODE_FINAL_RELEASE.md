# Restricted-Mode Final Release

## Purpose

Publish the final restricted-mode release package for AstraTreasury after governance decision recording, monitoring baseline, evidence seal, and release candidate readiness.

## Final release scope

- Recorded restricted-mode governance decision.
- Restricted-mode final public status.
- Post-governance monitoring baseline.
- Restricted-mode evidence seal.
- Restricted-mode release candidate.
- Capability Matrix.
- Public Status Update.
- Launch Control.
- Public site status pages.

## Current approved mode

Restricted Base Mainnet operation.

## Approved capabilities

None.

## Disabled / not approved

- Full launch.
- Public token sale.
- Real treasury funding.
- Staking or rewards.
- Buyback program.
- Mainnet execution queue activation.
- Paper-to-on-chain automation.
- Autonomous execution.
- Safe transaction payload generation.
- Safe transaction execution.

## Rule

The final release is a restricted-mode release artifact only. It does not activate or approve any capability.
