# DEX Liquidity Safe Execution Approval Boundaries

## This package can do

- Record approval for later Safe execution.
- Confirm threshold reached.
- Confirm the Safe transaction is still not executed.
- Publish execution approval status.

## This package cannot do

- Execute a liquidity Safe transaction.
- Add liquidity.
- Mint a position on-chain.
- Enable public trading.
- Activate the buy page.

## Rule

Approval only.
