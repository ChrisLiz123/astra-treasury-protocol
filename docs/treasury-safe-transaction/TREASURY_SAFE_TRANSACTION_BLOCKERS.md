# Treasury Safe Transaction Blockers

The following block Treasury Safe transaction preparation:

- Treasury disclosure not final-approved.
- Treasury risk limits not approved.
- Funding source not approved.
- Treasury Safe approval not recorded.
- Governance approval missing.
- Treasury funding not approved.
- Mainnet monitor not passing.
- Active incidents present.
- Destination not verified.
- Asset not verified.
- Amount not verified.
- Payload not decoded.
- Safe nonce not reviewed.

## Rule

No Treasury Safe transaction should be prepared while any blocker remains.
