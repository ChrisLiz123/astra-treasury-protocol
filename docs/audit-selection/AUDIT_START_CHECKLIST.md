# Audit Start Checklist

## Before audit starts

- [ ] Auditor selected.
- [ ] Statement of work agreed.
- [ ] Scope confirmed.
- [ ] Price confirmed.
- [ ] Timeline confirmed.
- [ ] Retest policy confirmed.
- [ ] Public report permission confirmed.
- [ ] Audit start commit frozen.
- [ ] Audit package uploaded.
- [ ] Communication channel created.
- [ ] Findings tracker ready.
- [ ] Remediation workflow ready.

## Technical readiness

- [ ] CI passing.
- [ ] release:prepare passing.
- [ ] audit:full passing.
- [ ] safe:prepare passing.
- [ ] mainnet runbook validation passing.
- [ ] public site online.
- [ ] docs current.

## Mainnet rule

Audit start does not authorize mainnet deployment.
