# Restricted-Mode Release Candidate

## Purpose

Create the release-candidate package for AstraTreasury restricted Base Mainnet operation after governance decision recording.

## Release candidate scope

- Recorded governance decision.
- Restricted-mode final public status.
- Post-governance monitoring baseline.
- Restricted-mode evidence seal.
- Capability Matrix.
- Public Status Update.
- Launch Control.
- Public site status pages.

## Current approved mode

Restricted Base Mainnet operation.

## Approved capabilities

None.

## Disabled / not approved

- Full launch.
- Public token sale.
- Real treasury funding.
- Staking or rewards.
- Buyback program.
- Mainnet execution queue activation.
- Paper-to-on-chain automation.
- Autonomous execution.
- Safe transaction payload generation.
- Safe transaction execution.

## Rule

The release candidate is a packaging and publication artifact only. It does not activate any capability.
