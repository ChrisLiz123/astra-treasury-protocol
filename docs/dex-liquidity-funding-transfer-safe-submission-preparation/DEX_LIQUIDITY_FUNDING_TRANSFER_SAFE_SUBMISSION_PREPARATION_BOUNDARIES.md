# DEX Liquidity Funding Transfer Safe Submission Preparation Boundaries

## This package can do

- Prepare submission instructions.
- Prepare transaction-builder fields.
- Prepare source Safe and payload references.
- Publish preparation evidence.

## This package cannot do

- Submit the Safe transaction.
- Execute the Safe transaction.
- Move treasury funds.
- Execute token approvals.
- Add liquidity.
- Mint a position.
- Enable public trading.
- Activate the buy page.

## Rule

Preparation only.
