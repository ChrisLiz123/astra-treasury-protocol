## Purpose

This document records the final review required before setting mainnetDeploymentApproved=true.

## Required before approval

- Audit cleared.
- Legal cleared.
- Production RPC configured.
- Backup RPC configured.
- Safe addresses finalized.
- Allocation wallets finalized.
- Deployer address finalized.
- Mainnet dry run v1 passed.
- Mainnet command generation v1 passed.
- Safe payload template generated.
- Incident response package complete.
- Governance gate reviewed.
- Final signer availability confirmed.

## Explicitly not approved by this checklist

- Public token sale.
- Real treasury funding.
- Staking or rewards launch.
- Buyback program.
- Autonomous execution.

## Rule

Do not set mainnetDeploymentApproved=true until this review is completed and signed off.
