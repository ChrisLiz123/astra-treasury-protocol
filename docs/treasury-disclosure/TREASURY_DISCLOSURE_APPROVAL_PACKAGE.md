# Treasury Disclosure Approval Package

## Current status

Treasury disclosure approval: NOT RECORDED

Treasury disclosure final approval: NO

Treasury funding: NOT APPROVED

Treasury funding transaction: NOT AUTHORIZED

## Purpose

Prepare the disclosure approval path required before any future treasury funding decision or transaction package.

## Required before disclosure approval

- Current public disclosure draft exists.
- Risk disclosure draft exists.
- Treasury funding source review exists.
- Treasury risk limits draft exists.
- Treasury Safe approval package exists.
- Treasury transaction dry run passed.
- Final disclosure reviewer assigned.
- Funding source approved, if funding is being proposed.
- Treasury risk limits approved, if funding is being proposed.
- Treasury Safe approval recorded, if a funding transaction is being proposed.
- Governance approval recorded.

## Rule

This package does not approve real treasury funding or any Safe transaction.
