# Treasury Disclosure Approval Blockers

The following block treasury disclosure final approval:

- Final disclosure reviewer not assigned.
- Restricted mode language not reviewed.
- No-public-sale language not reviewed.
- No-investment-product language not reviewed.
- Treasury funding language not reviewed.
- Risk disclosure not reviewed.
- Market data language not reviewed.
- Wallet visibility language not reviewed.
- Jurisdictional language not reviewed.
- Website pages not reviewed.
- API pages not reviewed.
- Final reviewer signoff not recorded.

## Rule

Disclosure approval must remain blocked until every required review item is complete.
