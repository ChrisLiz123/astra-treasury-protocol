# AstraTreasury Protocol

AstraTreasury Protocol is a Base Sepolia testnet prototype for AI-assisted treasury governance.

Status:

Version: v0.1
Network: Base Sepolia
Mainnet: not launched
Real treasury funds: no
Public sale: no
Investment product: no

Safety workflow:

AI paper signal -> manual signal approval queue -> SignalRegistry -> policy-aware execution queue -> manual execution approval -> ExecutionController -> TreasuryVault

AstraTreasury v0.1 is not an investment product and does not promise returns. Testnet ASTP has no real monetary value.
